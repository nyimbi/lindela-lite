# Security Policy

## Supported Versions

Security fixes target the latest `main` branch and tagged `v0.x` releases while Lindela Lite is pre-1.0.

## Reporting A Vulnerability

Do not open a public issue for vulnerabilities, exposed credentials, sensitive data leakage, or unsafe data-source behavior. Contact the maintainers privately or use GitHub Security Advisories when enabled.

Please include:

- A short description of the issue.
- Steps to reproduce.
- Affected version or commit.
- Whether secrets, client data, or public safety risks are involved.

## Data Safety

Do not submit client data, operational data, classified data, non-redistributable datasets, GDELT ingestion, or proprietary Lindela internals to this repository.
