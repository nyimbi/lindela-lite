# Open-Source Boundary

Lindela Lite is the public-good climate-conflict edition. It is useful on its own, but it preserves the commercial value of full Lindela by limiting the release to public ingestion, neutral schemas, baseline heuristics, a small API, and a lightweight UI.

## Open In Lite

- Public-source connectors for weather, disaster alerts, rainfall metadata, fire detections, flood feeds, and user-supplied service assets.
- Optional user-supplied conflict-event imports.
- Generic schemas for observations, hazard events, conflict events, service assets, impact assessments, source runs, and risk scores.
- Transparent baseline scoring for flood risk, climate-conflict pressure, and service-delivery impact.
- Local API and dashboard for inspection, export, and integration.

## Closed In Full Lindela

- Production ingestion orchestration, source lifecycle governance, enrichment, deduplication, source reputation, and data-quality systems.
- Calibrated climate-conflict and flood prediction models, trained coefficients, validation data, and proprietary feature engineering.
- Multi-INT fusion, classified reporting, report distribution, AAR workflows, wargaming, and enterprise collaboration.
- Client data, deployment automation, premium data connectors, classified workflows, and operational intelligence modules.

## Explicit Exclusions

- No Lindela commercial internals.
- No client data, secrets, or trained model artifacts.
